# FlyRank Assignments

This repository contains my assignments for the FlyRank program.

## Repository Structure

* `w2Assi1` – Week 2, Assignment 1

## Running `w2Assi1`

1. Navigate to the project directory:

```bash
cd w2Assi1
```

2. Install the dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

For testing through swagger ui
enter this url on browser - http://localhost:3000/docs
